## What are KPI targets?

**KPI targets** are the performance goals you set for each campaign. They define what success looks like—whether that's clicks, conversions, impressions, or custom metrics. Once you've created a campaign, you can adjust these targets anytime to reflect changing business needs or market conditions.

## When to edit KPI targets

You might edit KPI targets if:

- Campaign performance is tracking differently than expected
- Business priorities shift mid-campaign
- You want to set more realistic benchmarks based on early data
- You're running A/B tests and need to track different metrics

## How to edit KPI targets

1. Navigate to your campaign from the **Campaigns** dashboard
2. Click the **Settings** tab
3. Scroll to the **KPI Targets** section
4. Click **Edit** next to the KPI you want to change
5. Update the target value in the input field
6. (Optional) Adjust the **target date** if the deadline has moved
7. Click **Save**

## Editing multiple KPIs at once

If you need to update several targets:

1. In the **KPI Targets** section, click **Edit All**
2. Modify each target value as needed
3. Click **Save All** when finished

This approach saves time if you're recalibrating multiple metrics.

## Important notes

**Changes are retroactive in reporting.** When you edit a target, all historical performance data is re-evaluated against the new benchmark. Your campaign's status (on-track, at-risk, etc.) updates immediately.

**You can't delete KPIs after launch.** If you added a KPI when creating the campaign, you can change its target but can't remove it entirely. This keeps your historical data consistent. If you no longer want to track a metric, set its target to match current performance.

**Target changes appear in reports.** Team members viewing campaign reports will see the updated targets. There's no version history stored, so if you need to reference what the target was before, you'll want to note it separately.

**Dates and KPIs are linked.** If your campaign end date is May 31st, make sure your KPI target date doesn't extend beyond that. Campaignly will warn you if there's a mismatch.

## Troubleshooting

**Can't find the KPI Targets section?** Make sure you're in the campaign's **Settings** tab, not the **Overview** tab.

**Target won't save?** Check that your target value is a valid number and that you're not trying to edit a locked KPI (locked KPIs are set by your workspace admin and require admin permissions to change).

**Need to add a new KPI?** Unfortunately, you can't add KPIs after a campaign is created. Plan your metrics before you launch. For future campaigns, include all KPIs you'll want to track.
